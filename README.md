# Space Invaders 2D

A 2D Space Invaders clone built in Java using Swing, applying Object-Oriented Programming principles.

## About

Classic arcade-style shooting game where the player controls a spaceship and must destroy waves of incoming enemies before they reach the bottom of the screen. Features a real-time score system that tracks your performance throughout the game.

## Features

- Player-controlled spaceship with shooting mechanics
- Enemy waves with movement patterns
- Real-time score tracking
- Collision detection system
- Built with OOP design — separate classes for player, enemies, bullets, and game logic

## Tech Stack

- **Language:** Java
- **GUI Library:** Java Swing
- **Paradigm:** Object-Oriented Programming (OOP)

## How to Run

### Requirements
- Java JDK 8 or higher
- An IDE such as IntelliJ IDEA or Eclipse

### Steps
1. Clone the repository
   ```
   git clone https://github.com/Abdo-909156/Space-Invaders-game.git
   ```
2. Open the project in your IDE
3. Run the `Main.java` file (or whichever file contains the main method)

## Controls

| Key | Action |
|-----|--------|
| ← → Arrow Keys | Move spaceship left / right |
| Spacebar | Shoot |
